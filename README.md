# portfolioprojectfor100hires

This repository documents how this project was set up in Cursor and connected from GitHub.

## Tools installed

- **Cursor** — primary editor and AI-assisted development environment.
- **Extensions** — including **Claude Code** and **Codex** (and other extensions you added in Cursor).

## Steps completed

1. Created a **GitHub** repository and set it to **public**.
2. Enabled the **Add a README** option when creating the repository so GitHub generated an initial `README.md`.
3. After clicking **Create repository**, downloaded the repository as a **ZIP** file from GitHub.
4. Extracted the ZIP locally and **opened the folder in Cursor** to work on the project.
5. **Completed the README** (writing and editing the file), then **committed** and **pushed** the changes to GitHub.

## Issues encountered and how they were resolved

**Issue:** I was not sure how to open the README (and the repository folder) in Cursor at first.

**Resolution:** I searched online for how others open GitHub projects or folders in Cursor, followed a common approach (typically **File → Open Folder** and selecting the extracted project directory, or opening Cursor and choosing the folder that contains `README.md`), and confirmed the repository opens correctly in Cursor.
